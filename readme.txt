
1.运行环境为php环境，首页地址是：http://localhost/src/index.html
2.首页中，实现的效果有：轮播图，二级菜单，倒计时，楼层，缓动回到顶部，选项卡，手风琴效果，
3.登陆，注册功能都能实现，注册成功后跳转登陆页面，登陆成功之后跳转主页
4.在二级菜单中，”平板“是可以点击的商品列表，
5.点击进入后，第二行第一个是可以点击的商品详情页面，
6.每个商品下面的购买都是可以点击，增加到购物车内，
7.商品详情页面的加入购物车按钮，只是连接到购物车，并没有实际的增加的功能
8.购物车页面可以删除商品
9.商品详情页内数据为渲染数据，json数据
10.商品详情页面内的菜单为三级菜单，划过出现，隐藏
11.商品详情页面内左上，”电脑整机“和”电脑配件“都是可以点击出现内容列表	
12.每个页面的logo和”欢迎来新蛋购物！“都是可以点击返回主页面的
